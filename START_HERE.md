## 🆘 Need Help?

### **Website looks broken?**
- Hard refresh: `Ctrl + F5`
- Check browser console: `F12`

### **Cart not working?**
- Enable localStorage in browser
- Try different browser

### **Images not showing?**
- Check file names match exactly
- Verify files in `assets/images/`
